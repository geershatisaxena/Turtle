### Prerequisites

```bash
# Python 3.8 or higher required
python --version

# turtle ships with the standard library — nothing extra to install!
# Optional packages for enhanced visuals:
pip install Pillow colorsys
```

### Run Your First Project

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/Python-Turtle-Mega-Collection.git

# 2. Enter any category folder
cd Python-Turtle-Mega-Collection/05_Fractals/

# 3. Run any script — watch the magic unfold
python koch_snowflake.py
```
